## 项目简介

之前写过PlayAndroid：玩安卓的Compose版本，但最近在手机上看天气的时候发现一堆广告，烦得要死，自己写一个吧，正好再练练Compose。 之前写的是安卓版本的天气，这回搞一个桌面版的，让 `Mac`、`Windows`
、`Linux` 都能使用！废话不多说，先来看看项目运行效果吧！
